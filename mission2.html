Le context

Le but de cette mission est de découvrir la base de données kdou et de se familiariser avec elle pour savoir ce qui est important et ce qui ne l'est pas en utilisant le logiciel low code Access Le thème de la mission (ce que nous étudions par cette analyse) est l'étude de marché du marché mondial en général et de quelques échantillons spécifiques (acheteurs). Nous voulons savoir quels produits sont populaires ? dans quels endroits ? quels produits sont chers ? et quels produits ne le sont pas ? quels pays offrent le plus de produits ? quels pays ont le plus de clients ?

indicateurs intéressants

    • Quels sont les produits d’Italie ou d’Allemagne vendus 50€ ou moins ? (On veut les connaître noms et prix, nom du fournisseur, ville et pays d’origine.) 
      • SELECT Produit.NomProd, Produit.PrixUnit, Four-nisseur.Contact, Fournisseur.Ville, Fournisseur.Pays
FROM Fournisseur INNER JOIN Produit ON Fournisseur.NoFour = Produit.NoFour
WHERE (((Produit.PrixUnit)<50) AND ((Fournis-seur.Pays)="Italie" Or (Fournisseur.Pays)="Allemagne"));
    • Quelles sont les catégories de produit vendues par les four-nisseurs Singapouriens ? 
      • SELECT Categorie.NomCateg, Fournisseur.Pays
FROM Fournisseur INNER JOIN (Categorie INNER JOIN Produit ON Categorie.CodeCateg = Produit.CodeCateg) ON Fournisseur.NoFour = Produit.NoFour
WHERE (((Fournisseur.Pays)="Singapour"));
    • Combien y a-t-il de clients nantais 
      • SELECT Client.CodeCli, Client.VilleCli
FROM Client
WHERE (((Client.VilleCli)="Nantes"));
    • Donnez l’offre de produits suédois 
      • SELECT Produit.NomProd, Fournisseur.Pays
FROM Fournisseur INNER JOIN Produit ON Fournisseur.NoFour = Produit.NoFour
WHERE (((Fournisseur.Pays)="Suède" Or (Fournisseur.Pays)="Suéde"));
    • Donner le prix moyen de vente des produits, le prix minimum et le prix maximum. 
      • SELECT Max(Produit.PrixUnit) AS MaxDePrixUnit, Min(Produit.PrixUnit) AS MinDePrixUnit, Avg(Produit.PrixUnit) AS MoyenneDePrixUnit
FROM Produit;
    • Quelle est la fonction de contact fournisseur la plus représentée ? 
      • SELECT Fournisseur.Fonction, Count(Fournisseur.Fonction) AS CompteDeFonction
FROM Fournisseur
GROUP BY Fournisseur.Fonction
ORDER BY Count(Fournisseur.Fonction) DESC;
    • Quels sont les produits conditionnés dans du carton ou dans du verre ? 
      • SELECT Produit.NomProd, Produit.QteParUnite
FROM Produit
GROUP BY Produit.NomProd, Produit.QteParUnite
HAVING (((Produit.QteParUnite) Like "*verre*" Or (Produit.QteParUnite) Like "*carton*"));
    • Donner pour chaque fournisseur le nombre de produits qu'il propose en bouteille 
      • SELECT Fournisseur.Societe, Count(Produit.QteParUnite) AS QteEnBouteille
FROM Fournisseur INNER JOIN Produit ON Fournisseur.NoFour = Produit.NoFour
WHERE (((Produit.QteParUnite) Like "*bouteille*"))
GROUP BY Fournisseur.Societe;
    • On veut le chiffre d’affaire total généré avec les informations dont on dispose. 
      • SELECT Sum(PrixUnitCli*QteCom) AS Chiffredaffaire
FROM DetailCommande;
    • Est-ce qu’il y a dans la clientèle des clients qui n’ont pas encore passé de commande ? 
      • SELECT Client.CodeCli
FROM Client LEFT JOIN Commande ON Client.CodeCli = Commande.CodeCli
GROUP BY Client.CodeCli, Commande.CodeCli
HAVING (((Commande.CodeCli) Is Null));
    • Dans quelles villes la clientèle est-elle la plus importante ? Citer les 3 premières. 
      • SELECT TOP 3 Client.VilleCli, Count(Client.CodeCli) AS CompteDeCodeCli
FROM Client
GROUP BY Client.VilleCli
ORDER BY Count(Client.CodeCli) DESC;
